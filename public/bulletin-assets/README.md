# Bulletin Assets

Este directorio contiene las imágenes y assets para el diseño clásico del boletín.

## Archivos Requeridos

### header-bg.png
- Dimensiones: 1024px x 234px
- Descripción: Imagen de fondo del header (la imagen roja con el logo)
- Formato: PNG, JPG o WebP
- Ubicación: public/bulletin-assets/header-bg.png

## Instrucciones

1. Coloca la imagen del header en este directorio
2. Renómbrala a header-bg.png
3. El componente ClassicBulletinV2 la usará automáticamente
